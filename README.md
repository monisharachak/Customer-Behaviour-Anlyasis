# Customer Behavior Analysis Dashboard | Power BI



## Overview

This project focuses on analyzing customer behavior using an interactive Power BI dashboard. The dashboard helps businesses understand customer purchasing patterns, engagement levels, and demographic trends to support better business decisions.

## Tools Used

* Power BI  * Power Query * DAX * Data Modeling * Data Cleaning and Transformation
## Objectives

* Analyze customer purchasing behavior and preferences.
* Identify valuable customer segments.
* Track customer retention and engagement.
* Monitor sales and revenue performance.
* Generate actionable business insights through data visualization.

## Key Analysis

### Customer Analysis

* Purchase frequency and buying patterns
* New vs. returning customers
* Customer segmentation based on spending
* Customer retention and loyalty analysis
* Product preference analysis

### Demographic Analysis

* Age group distribution
* Gender-based purchasing trends
* Geographic customer distribution

### Sales Analysis

* Revenue by customer segment
* Top customers by sales contribution
* Average order value
* Monthly and yearly sales trends

## Dashboard Features

* Interactive Power BI reports
* Dynamic filters and slicers
* KPI cards and performance metrics
* Drill-down analysis
* Customer segmentation visuals
* Trend and geographic analysis



## Key Insights

* Identified high-value customer segments.
* Analyzed customer retention and repeat purchase behavior.
* Discovered demographic groups contributing the most revenue.
* Tracked sales performance and customer growth trends.
* Supported data-driven business and marketing decisions.

## Skills Demonstrated

* Data Analysis
* Business Intelligence
* Power BI Dashboard Development
* Data Visualization
* DAX
* Customer Analytics
* KPI Reporting
* Data Modeling

## Project Outcome

Built a comprehensive Power BI dashboard that provides meaningful insights into customer behavior, sales performance, and customer engagement, helping businesses improve decision-making and customer retention strategies.
